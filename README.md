# Proyecto 1 – Análisis Exploratorio de Datos de Ventas

## Descripción
Este proyecto consiste en un Análisis Exploratorio de Datos (EDA) aplicado a un conjunto de datos de ventas de una tienda.
El objetivo es analizar el comportamiento de las ventas, identificar los productos más vendidos, evaluar los ingresos generados y estudiar la distribución de precios a lo largo del tiempo.

## Dataset
El dataset es de tipo simulado y contiene 500 registros de ventas con las siguientes columnas:

- ID_Venta: identificador único de la venta
- Producto: producto vendido (Laptop, Mouse, Teclado, Monitor, Audífonos)
- Categoria: tipo de producto (Electrónica o Accesorios)
- Precio: precio unitario del producto
- Cantidad: cantidad vendida
- Fecha: fecha de la venta
- Total: ingreso total por venta (Precio x Cantidad)

## Tecnologías Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Análisis Realizado
Durante el análisis se realizaron las siguientes tareas:

- Exploración inicial del dataset (estructura, tipos de datos y valores nulos)
- Estadísticas descriptivas de las variables numéricas
- Análisis de ventas por producto
- Análisis de ingresos totales por producto
- Cálculo del precio promedio por producto
- Análisis por categoría
- Análisis temporal de ventas por mes
- Cálculo de métricas generales del negocio
- Análisis de correlación entre variables numéricas

## Visualizaciones
El proyecto incluye las siguientes visualizaciones:

- Gráfica de barras: cantidad de ventas por producto
- Gráfica de barras horizontales: ingresos por producto
- Histograma: distribución de precios con media y mediana
- Gráfico de dispersión: relación entre precio y total de venta
- Gráfica de línea: evolución de ventas por mes
- Dashboard con múltiples gráficos
- Heatmap de la matriz de correlación

## Resultados Principales
- Se identificaron los productos con mayor número de ventas.
- Se analizaron los productos que generan mayores ingresos.
- La distribución de precios muestra una concentración en ciertos rangos.
- Existe una relación positiva entre el precio y el total de venta.
- Se observaron variaciones en las ventas a lo largo de los meses.
